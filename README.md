# Kanshan NEO

Kanshan NEO 是一个围绕「赛博刘看山」构建的软硬件原型：

它把刘看山装进一块低功耗墨水屏里，

让它成为能陪你创作、提问、出行打卡和轻度社交的电子宠物。

```text
你的下一个电子好友，可以是看山~
```

项目站点：[https://kanshan.r2049.cn/](https://kanshan.r2049.cn/)

## 核心能力

Kanshan NEO 目前围绕 5 个方向展开：

- **旅行刘看山**：知乎创作驱动，解锁图章和只属于你的行记。
- **看山百晓生**：支持知乎直答，大模型连续对话。
- **创作助手**：感兴趣话题一键推送，创作数据随时看。
- **离线养成**：低功耗墨水屏，电池驱动，随时可用。
- **轻度社交**：无网近场消息、在线聊天室，和看山一起结识朋友。

## 仓库结构

- `esp32/`：ESP32-S3 知乎助手固件，包含 `zhihu_assistant` 设备逻辑、墨水屏驱动、按键、音频和网络模块。
- `kanshan-server/`：FastAPI 后端和 Web 前端，提供设备 HTTP 接口、登录、应用页面和管理能力。
- `browser-extension/`：浏览器插件，用于辅助创作场景和浏览器联动。
- `game/`：刘看山游戏素材和角色资源，Web 首页中的卡片插图来自这里的素材子集。
